Eco Hero 33 - 2D RPG Game

🌍 What is it?

Eco Hero 33 is a 2D Role-Playing Game developed entirely in C using the SDL2 library. Set in a polluted post-apocalyptic world in the year 2333, the player must leave their bunker to restore the ecosystem by fighting mutant viruses, correctly recycling waste, and planting trees.

🛠️ Technologies Used

Language: C

Graphics & Inputs: SDL2 (Simple DirectMedia Layer), SDL_ttf

Build Tool: Makefile / GCC

✨ Key Features

Custom Game Engine: Hand-coded matrix-based map rendering and scene transitions.

Physics & Collisions: Custom hitbox-based collision detection for environment and entities.

Interactive Inventory: A drag-and-drop inventory system to manage items, weapons, and seeds.

Enemy AI: Basic pathfinding and pursuit logic for enemy interactions.
